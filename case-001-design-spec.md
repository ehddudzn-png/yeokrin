# 추리게임 — 사건 설계 명세서 #001

> 이 문서는 **구현용 사양서**다. Claude Code(VS Code)에 이 파일을 통째로 주고
> "이 명세대로 단일 HTML(또는 React) 추리게임을 구현해줘"라고 하면 된다.
> 스포일러가 가득하니 **플레이어에게는 절대 노출 금지**. 개발자(너)만 본다.

---

## 0. 설계 철학 (이걸 어기면 게임이 망한다)

1. **제목·메뉴·UI 어디에도 트릭 유형을 적지 않는다.** 사건명은 무미건조하게(`사건 #001`, 또는 분위기용 중립 제목 `늦은 비`). "서술트릭" "1인2역" 같은 단어는 코드 주석에만.
2. **객관식 보기에 답의 힌트를 절대 넣지 않는다.** "나(화자)" 같이 튀는 보기 금지. 모든 선택지는 평범하고 그럴듯해야 한다.
3. **찍기 불가능.** 범인 하나만 고르는 게 아니라 `범인 + 수법 + 결정적 모순` 3개를 동시에 맞춰야 종결. 하나라도 틀리면 "논리에 구멍이 있다"며 반려 + 어느 축이 틀렸는지만 힌트.
4. **페어플레이.** 정답에 필요한 모든 단서는 플레이어가 접근 가능해야 한다. 숨긴 정보로 뒤통수치지 않는다. 트릭은 "정보를 숨겨서"가 아니라 "정보를 다르게 해석하게 유도해서" 성립한다.
5. **능동적 조사.** 단순히 "다음" 누르기 금지. 현장의 지점을 직접 탭해 조사하고, 진술을 직접 펼치고, 증거끼리 대조하는 행위가 있어야 한다.

## 1. 사건 개요 (플레이어가 보는 표면)

**사건 #001 — 「늦은 비」**

> 밤 11시, 빗속. 오래된 단층 주택의 거실에서 **남자(피해자)**가 가슴을 찔린 채 발견됐다.
> 신고자는 **동거인 여성**. "샤워하고 나왔더니 쓰러져 있었다"고 한다.
> 집은 안에서 잠겨 있지 않았고, 비 때문에 외부 발자국은 쓸려나갔다.
> 용의자는 셋: 동거인 여성, 피해자의 사업 동업자, 옆집 남자.

표면적 사건은 **평범한 치정/금전 살인**처럼 보이게 한다. 트릭이 있다는 낌새를 주지 않는다.

---

## 2. 진짜 진상 (THE TRUTH — 절대 노출 금지)

**피해자라고 알려진 시신은 사실 "사업 동업자"다. 그리고 신고자 여성이 "동거인"이라고 부른, 살아서 사라진 "피해자 남자"가 진범이다.**

- 즉 **시신의 신원이 뒤바뀌어 있다.** (서술/인지 트릭: 플레이어는 처음부터 "시신 = 동거남(피해자)"이라고 믿게 유도된다. 아무도 그걸 명시적으로 거짓말하지 않는다 — 단지 그렇게 보이게 배치됐을 뿐.)
- 진범(원래의 "피해자 남자", 이름: **강무진**)은 빚에 쫓기고 있었다. 사업 동업자 **윤태식**에게 거액의 채무 + 보험. 강무진은 자기 명의로 거액 생명보험을 들어두고, **체격이 비슷한 동업자 윤태식을 살해해 자신으로 위장**한 뒤 사라졌다. 보험금 + 빚 청산 + 새 삶.
- 신고자 여성 **서연희**는 강무진의 동거인. 처음엔 공범인지 피해자인지 모호하게 두되 — 진실은 **그녀도 속았다(진짜 피해자 측)**. 강무진이 그녀까지 이용했다. → 마지막 반전의 정서적 무게.

**왜 "와 미쳤다"인가:** 플레이어는 내내 "동거남이 죽었다, 누가 죽였나"를 수사한다. 진짜 질문은 **"죽은 사람이 정말 그 사람이 맞나?"**였다. 범인을 찾는 게 아니라 **시신의 정체를 의심**해야 풀린다. 모든 단서는 처음부터 신원 바꿔치기를 가리키고 있었다.

## 3. 등장인물

| ID | 이름 | 표면 | 진실 |
|----|------|------|------|
| `victim` | (시신) | "동거남 강무진"으로 추정됨 | 실제로는 **동업자 윤태식** |
| `yh` | 서연희 | 신고자·동거인 | 진짜 피해자(속은 자). 거짓말 안 함 — 그녀도 시신을 강무진으로 오인 |
| `ts` | 윤태식 | 사업 동업자 (행방 묘연) | **실제 시신 본인** (그래서 "연락 두절") |
| `op` | 오필립 | 옆집 남자·목격자 | 무관(레드헤링). 단 결정적 목격 1개 보유 |
| `mj` | 강무진 | "피해자"로 알려짐 (시신으로 간주) | **진범. 살아있음.** 도주 중 |

핵심 트릭 장치: **강무진과 윤태식은 체격·나이가 비슷**하고, 시신은 **얼굴이 둔기로 심하게 손상**돼 육안 식별 불가. 신원은 "이 집에 살던 사람 + 옷 + 소지품"으로 추정됐을 뿐 — 누구도 지문/DNA로 확정하지 않았다.

## 4. 증거 목록 (플레이어가 조사로 획득)

각 증거는 `id / 표면 의미 / 진짜 의미(트릭 단서인지)`로 정리.

| ID | 이름 | 표면 의미 | 진짜 단서 |
|----|------|-----------|-----------|
| `ring` | 시신 손가락의 결혼반지 | "동거남의 반지" | **반지 안쪽 각인이 'T'**. 서연희는 강무진과 결혼 안 함(동거). 동업자 윤태식은 기혼 → 시신은 윤태식 |
| `watch` | 시신의 손목시계 | 평범한 시계 | **윤태식이 늘 차던 브랜드/모델**(동업자 인터뷰·사진에서 확인 가능). 강무진은 시계 안 참(서연희 진술) |
| `face` | 검시 소견 | "둔기로 얼굴 손상, 가슴 자상이 사인" | **굳이 얼굴을 으깬 이유** = 신원 위장. 사인은 가슴인데 얼굴까지 친 건 과잉 → 식별 방해 의도 |
| `insurance` | 강무진 명의 생명보험 | "강무진 앞 거액 보험" | 수익자: 서연희가 아님 → **제3의 계좌(강무진 본인이 통제하는 차명)**. 죽은 자가 자기 보험금을 탄다는 모순 |
| `debt` | 채무 기록 | "강무진이 윤태식에게 큰 빚" | 강무진이 죽으면 빚은 소멸·보험금 발생. **죽어서 이득 보는 건 강무진 본인** |
| `cctv` | 인근 편의점 CCTV | "사건 후 비옷 입은 인물" | 사건 직후 **시신과 같은 체격의 남자가 멀쩡히 걸어나감**(11:20). 죽은 사람이 걸을 순 없다 → 시신은 다른 사람 |
| `phone` | 시신 곁 휴대폰 | "강무진 폰" | **윤태식에게 온 부재중 전화가 없다** — 동업자가 죽었다면 강무진이 찾았을 텐데. 오히려 강무진 폰엔 윤태식과의 마지막 통화가 삭제됨 |
| `op_see` | 오필립의 목격담 | "비 오는데 누가 우산 없이 나갔다" | 11:20경 **그 집에서 한 남자가 멀쩡히 나와 차를 몰고 떠남**. 서연희는 그 시간 집 안(샤워). cctv와 교차 검증 |

**레드헤링(함정):** `op`(옆집 남자)는 피해자와 사소한 시비가 있었음 → 동기처럼 보임. 하지만 그의 목격담이 오히려 진실의 열쇠. 플레이어가 "동기 있는 옆집남"으로 새기 쉽게.

## 5. 진술 (심문) — 어디가 거짓이고 어디가 진실인가

**원칙: 거짓말하는 인물은 강무진(부재)뿐. 나머지는 다 진실을 말한다 — 단지 "시신=강무진"이라는 전제를 모두가 공유해서 진실이 오해를 부른다.**

### 서연희 (`yh`) — 전부 진실
- "그이(강무진)는 시계를 안 차요. 답답해해서." → (시신은 시계를 차고 있다 = 모순 단서)
- "우린 결혼은 안 했어요. 그이가 미루자고 해서." → (시신 손가락엔 결혼반지 = 모순)
- "샤워하고 나오니 쓰러져 있었어요. 11시 좀 넘어서." → 알리바이 진실

### 오필립 (`op`) — 진실 (단 본인은 의미를 모름)
- "11시 20분쯤, 그 집에서 남자가 나와 차 몰고 갔어요. 비 오는데 멀쩡히." → 결정적
- "죽은 사람이랑 나랑 주차 문제로 다툰 적은 있어요." → 레드헤링(동기로 보이게)

### "윤태식"에 대해 (부재중) — 주변 진술
- 서연희: "그이 동업자 윤태식 씨요? 며칠째 연락이 안 된대요." → (시신이 윤태식이라서 연락 안 됨)
- 회사 직원 메모: "윤 대표, 그날 저녁 강무진 집에 간다고 나가서 안 돌아옴." → 시신=윤태식 결정타

### 강무진 (`mj`) — 부재(시신으로 간주). 그가 남긴 흔적이 곧 거짓
- 삭제된 통화기록, 차명계좌 보험 수익자 설정 = 살아있는 자의 공작

## 6. 정답 판정 로직 (찍기 불가 — 3축 동시 정답)

플레이어는 종결 시 **세 가지를 각각 지정**해야 한다. (객관식 아님 — 아래 방식 권장)

1. **`죽은 사람은 누구인가`** → 정답: `윤태식` (동업자)
   - 후보 제시 방식: "시신의 신원"을 용의자/관계자 명단에서 고르게. "강무진(표면 답)"도 평범하게 섞어둬서, 단서 없이 고르면 틀리게.
2. **`범인은 누구인가`** → 정답: `강무진` (= 죽은 줄 알았던 사람)
3. **`결정적 증거(트릭을 증명하는 것)`** → 정답: `cctv` 또는 `ring` 중 핵심 1개로 한정. (구현 시 둘 다 정답 처리하거나, "신원"을 증명하는 `ring`을 정답으로)

**판정:**
- 셋 다 정답 → 진상 공개(아래 7번).
- `죽은 사람`이 틀림 → "당신은 아직 '누가 죽었는지'를 의심하지 않았다. 시신의 신원을 확정한 사람이 있었던가?" (가장 깊은 힌트)
- `죽은 사람`은 맞고 `범인`이 틀림 → "시신이 동업자라면, 강무진은 어디 있는가. 죽어서 이득 보는 자는 누구인가."
- `증거`만 틀림 → "방향은 맞다. 하지만 신원 바꿔치기를 '증명'하는 단서는 정황이 아니라 신체에 있다(반지/시계/CCTV)."

> **찍기 차단 핵심:** 1번(시신의 신원)을 의심하는 발상 자체가 트릭의 정체라서, 단서를 안 보면 절대 못 맞춘다. "범인 누구?"만 물으면 찍힐 수 있으니 **"누가 죽었나"를 첫 질문으로** 두는 게 이 사건의 자물쇠.

## 7. 진상 공개 텍스트 (정답 시 — 이 톤으로)

> 당신은 처음부터 "강무진이 죽었다"고 믿었다. 경찰도, 서연희도, 당신도.
> 하지만 그 시신의 신원을 **확정한 사람은 아무도 없었다.** 얼굴은 으깨졌고, 신원은 "이 집에 살던 사람"이라는 이유로 추정됐을 뿐이다.
>
> 시신은 강무진이 아니라 **동업자 윤태식**이었다. 손가락의 결혼반지(강무진은 미혼·동거), 늘 차던 시계(강무진은 시계를 안 찼다), 그리고 11시 20분 그 집에서 멀쩡히 걸어나와 차를 몬 남자 — 죽은 사람은 운전하지 않는다.
>
> 강무진은 빚에 몰려 있었다. 자기 명의로 거액의 보험을 든 뒤, 체격이 비슷한 윤태식을 불러 살해하고 얼굴을 으깨 **자신으로 위장**했다. 그는 죽지 않았다. 자신의 장례를 연출하고 보험금을 챙긴 채 사라졌다.
>
> 그리고 서연희 — 그녀는 공범이 아니었다. 가장 가까이서 가장 깊게 속은 사람이었다.
>
> 우리는 "누가 강무진을 죽였나"를 물었다. 진짜 질문은 — **"죽은 사람이 정말 강무진인가"**였다.

오답 정서 포인트: 마지막에 서연희가 "그이가… 살아있다고요?" 하는 한 컷.

## 8. 플레이 흐름 (씬 구조)

```
[메뉴] 사건 #001 「늦은 비」  ← 트릭 암시 0
   ↓
[S1 현장] 빗속 거실, 시신.  → 탭 조사: 반지 / 시계 / 얼굴(검시) / 휴대폰
   ↓ (4개 다 조사해야 진행)
[S2 심문·서연희] 진술 3개 펼쳐 읽기 (시계·결혼·알리바이)
   ↓
[S3 심문·오필립] 목격담 + 레드헤링 동기
   ↓
[S4 자료] 보험·채무·CCTV·회사메모 → 탭 조사
   ↓
[S5 종결] 3축 지정(누가 죽었나 / 범인 / 결정적 증거) → 판정
```

**능동성 장치:** S1·S4는 모든 지점 조사해야 다음 해금. 진술은 접힌 걸 탭해서 펼침. 증거끼리 "대조" 기능(반지+서연희진술 = 모순 하이라이트) 넣으면 더 좋음.

```js
  const okDead = pick.deadIs === SOLUTION.deadIs;
  const okCul  = pick.culprit === SOLUTION.culprit;
  const okEv   = SOLUTION.keyEvidence.includes(pick.keyEvidence);
  if(okDead && okCul && okEv) return {win:true};
  if(!okDead) return {win:false, hint:'시신의 신원을 의심하라...'};
  if(!okCul)  return {win:false, hint:'죽어서 이득 보는 자는...'};
  return {win:false, hint:'신원을 증명하는 단서는 신체에 있다...'};
}
```

## 10. 페어플레이 체크리스트 (구현 후 자가검증)

- [ ] 정답(`시신=윤태식`)을 가리키는 단서가 **최소 3개**(반지·시계·CCTV) 모두 플레이어 접근 가능한가
- [ ] 어떤 인물도 "시신은 강무진이다"라고 **명시적으로 거짓 단언**하지 않는가 (다들 '추정/오인'일 뿐)
- [ ] 메뉴·제목·보기에 트릭 힌트가 0인가
- [ ] 종결 보기에 "튀는 정답"이 없는가 (강무진/윤태식이 둘 다 평범하게 섞여 있는가)
- [ ] 단서 안 보고 찍으면 1/N로 틀리게 되는가 ("죽은 사람"을 첫 질문으로 둬서 잠금)
